## 4. Value Chain Flow

```
RAW MATERIALS → PROCESSING → TRANSPORTATION → DISTRIBUTION → END USER

Fossil Fuel Path:
─────────────────
Crude Oil/Natural Gas → Extraction → Pipeline/Tanker → Refining → 
Petroleum Products → Distribution → Gas Stations/Industry

Renewable Energy Path:
─────────────────────
Sun/Wind/Water → Harvesting (Solar panels, Turbines) → 
Inverter/Transformer → Transmission → Distribution → 
Industrial/Commercial/Residential

Electricity Value Chain:
─────────────────────────
Generation (Power Plant) → Transmission (High Voltage) → 
Substation → Distribution (Low Voltage) → End User

Emerging:
─────────
Raw Materials (Lithium, Cobalt) → Battery Manufacturing → 
Packaging → Grid Storage/EVs → Recycling
```
