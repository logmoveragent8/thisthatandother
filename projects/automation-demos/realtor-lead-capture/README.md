# Realtor Lead Capture Workflow

## Overview
Automates new property inquiries and lead follow-up for real estate agents

## Use Cases

### 1. Property Inquiry Bot
- Buyer asks about a property
- Details saved to CRM (Google Sheets)
- Auto-reply with property info
- Agent gets Telegram alert

### 2. Home Valuation Request
- Homeowner wants to sell
- Collects: Address, Phone, Email, bedrooms, bathrooms
- Agent notified immediately

### 3. Open House Registration
- Sign-in at open house
- Auto-add to follow-up sequence
- Send info about similar properties

---

## Value for Realtor

| Problem | Solution |
|---------|----------|
| Leads lost in texts | Centralized in Sheets |
| Slow follow-up | Instant Telegram alert |
| No info on prospects | Auto-capture all details |
